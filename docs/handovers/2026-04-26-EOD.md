## Architecture Overview

### Deployment Pipeline
- GitHub repo: paddygallivan/kbt-trial
- Auto-deploy: every push triggers gh-push Worker which writes files to
  KBT_ADMIN KV namespace AND pushes to CF Pages
- CF Pages URL: kbt-trial-9gu.pages.dev (player-app.html lives here)
- CF Worker: kbt-admin (account a6f47c17) serves host-app + admin-app
  from KBT_ADMIN KV, plus handles /api/question endpoint

### Supabase Tables
- kbt_events: event registry
- kbt_teams: admin-registered teams
- trial_registrations: self-registered teams
- kbt_live_answers: live answer submissions
- trial_scores: round score upserts
