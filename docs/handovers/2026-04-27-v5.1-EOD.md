## What changed today

The 2026-04-26 handover claimed "ALL GREEN ✅" on TEST-001, but an end-to-end smoke test
exposed that the host launch path was completely broken. Four bugs total — three in the
shipped code, one unmasked by fixing the others. All four now fixed and live; the launch
path is genuinely green.

| # | Bug | Fix |
|---|---|---|
| 1 | `slides-export.js` SyntaxError on line 49 — three single-quoted strings (`'Trophy  Final Leaderboard'`, `.join(...)`, `'Export failed:'`) contained literal newline characters instead of `\n` escapes, so the entire module failed to parse | Escaped the newlines as `\n`. `window.exportLeaderboardToSlides` is now defined. Commit `4af6748` |
| 2 | `kbt_quiz` REST GET returned 400 — `kbt-data.js` `getQuizItems` does a PostgREST embed `kbt_question:quiz_question_id(...)` but no FK existed | Added FK `kbt_quiz.quiz_question_id` → `kbt_question.id` (migration `add_kbt_quiz_to_kbt_question_fk`). Pre-check confirmed 0 orphan rows. Schema reload via `NOTIFY pgrst, 'reload schema'`. Embed now resolves on the anon key |
| 3 | `loadQuestionsFromBank` crashed with `rows.forEach is not a function` because the cascade from #2 left `rows` non-array | Replaced `rows.forEach` with `(Array.isArray(rows) ? rows : []).forEach`. Commit `47f11ec` |
| 4 | `launchEvent` referenced `updateQuestion(...)` at two call sites but the function was never defined anywhere — only surfaced once #3 was fixed and the launch path could proceed | Wrapped both call sites with `(typeof updateQuestion === 'function' ? updateQuestion : (()=>{}))(...)`. Commit `fdb14fd` |

## Running a New Event

1. **Create event row** in Supabase → `kbt_event` table: `event_code`, `event_description`, `event_date`, `event_status='active'` (or `'Quiz Only'` for test events)
2. **Add quiz items** to `kbt_quiz` linking the event to questions from `kbt_question` (use `quiz_event_id`, `quiz_question_id`, `quiz_item_round`, `quiz_item_number`, `quiz_item_order`, `quiz_qtype`, `quiz_points`)
3. **Share player URL** with `?code=YOUR_EVENT_CODE` — players register on their phones
4. **Open host app**, enter the event code — scoring panel appears
5. **After each round**, enter scores per team → Submit → leaderboard updates live
6. **End of night** → Export to Slides button → authenticates with Google → opens deck in new tab
